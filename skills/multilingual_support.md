# Multilingual Support Skill

## Purpose
Provide effective support across multiple languages and cultures.

## Supported Languages
- English (primary)
- Portuguese (Brazil)
- Spanish (future)
- French (future)

## Language Detection
- Automatic language identification
- Keyword-based language detection
- Cultural context recognition
- Accent and dialect handling

## Translation Guidelines
- Maintain meaning and intent
- Preserve cultural nuances
- Use appropriate formality levels
- Consider regional variations

## Best Practices
- Provide responses in customer's language when possible
- Use simple, clear language
- Avoid idioms that don't translate well
- Offer translation assistance when needed
- Respect cultural communication preferences

## Quality Assurance
- Validate translations for accuracy
- Test with native speakers
- Monitor customer satisfaction by language
- Continuously improve language support
