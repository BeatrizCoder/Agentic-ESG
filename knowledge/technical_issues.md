# Technical Issues

## Website or App Not Working
Steps to try:
1. Refresh the page (F5)
2. Clear browser cache and cookies
3. Try a different browser or device
4. Check internet connection
5. Disable browser extensions temporarily
If widespread: check status.supportai.com
Contact support with: browser version, device, OS, screenshot
Escalate if: widespread outage or user has critical pending order

## Checkout or Payment Error
- Verify card details, billing address, available balance
- Try a different payment method
- Contact your bank (some block online transactions)
Common error codes:
- ERR_PAYMENT_DECLINED: card declined by bank
- ERR_CVV_MISMATCH: security code incorrect
- ERR_EXPIRED_CARD: card expired
Escalate immediately if: charged but order not confirmed
