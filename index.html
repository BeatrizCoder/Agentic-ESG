      Política de Privacidade — Agentic ESG
      1. Data We Collect / Dados que Coletamos

      We collect no personal data. Only an anonymous session ID (UUID) is stored as a browser cookie for 30 days to enable analysis history.

      Não coletamos dados pessoais. Apenas um ID de sessão anônimo (UUID) é armazenado como cookie por 24 horas para habilitar o histórico de análises.

      2. Data Retention / Retenção de Dados

      All session data is automatically deleted after 30 days via MongoDB TTL index. No backups are kept. No data is sold or shared with third parties.

      Todos os dados de sessão são automaticamente excluídos após 24 horas via índice TTL do MongoDB. Nenhum backup é mantido. Nenhum dado é vendido ou compartilhado.

      3. Third-Party Services / Serviços de Terceiros

      NASA POWER API, OpenMeteo, Anthropic Claude API, and Railway are used to deliver the service. None of these services receive your personal data.

      NASA POWER API, OpenMeteo, Anthropic Claude API e Railway são utilizados para entregar o serviço. Nenhum desses serviços recebe seus dados pessoais.

      4. Your Rights / Seus Direitos (LGPD / GDPR)

      You may decline cookies at any time. Declining disables analysis history but does not affect core functionality.

      Você pode recusar os cookies a qualquer momento. Recusar desabilita o histórico de análises, mas não afeta a funcionalidade principal.
